## 创建任务方法

:::warning **注意！**
在求解过程开始时，可能会出现临时的 *unsolvable*（不可解）响应。这**不是错误**——初始化完成后验证码仍会继续成功解决。
:::

<TabItem value="proxyless" label="ComplexImageTask（无代理）" default className="method-panel">
	<MethodItem>
		```http
		https://api.capmonster.cloud/createTask
		```
	</MethodItem>
	<MethodDescription>
      **请求**
      ```json
      {
        "clientKey": "API_KEY",
        "task": 
        {
          "type": "ComplexImageTask",
          "class": "recognition",
          "imagesBase64": ["base64"],
          "metadata": {
            "Task": "baidu"
          }
        }
      }
      ```

    	任务示例：

    	![](../images/ComplexImageTask-Recognition/baidu.png)

        **响应**
        ```json
        {
          "errorId":0,
          "taskId":143998457
        }
        ```
    </MethodDescription>

</TabItem>

## 如何获取 Base64

页面中的图片可以以 URL 形式存在，也可以已经被编码为 Base64 格式。要找到所需的值，请在验证码图片上右键，选择 **检查（Inspect）**，然后仔细查看 **Elements（元素）** 面板或网络请求 —— 在那里你可以找到图片 URL 或已编码的内容。

1. 在浏览器中打开显示验证码的网站。  
2. 右键点击验证码元素并选择 **检查（Inspect）**。
