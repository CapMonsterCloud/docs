## еҰӮдҪ•иҺ·еҸ–еҲӣе»әд»»еҠЎжүҖйңҖзҡ„жүҖжңүеҸӮж•°

### `sceneId`

`sceneId` еҸҜд»ҘеңЁжҲҗеҠҹе®ҢжҲҗдёҖж¬Ў CAPTCHA еҗҺиҺ·еҸ–пјҡ

1. еңЁзҪ‘з«ҷдёҠжүӢеҠЁе®ҢжҲҗ CAPTCHAгҖӮ
2. жү“ејҖ **DevTools** вҶ’ **Network** ж ҮзӯҫгҖӮ
3. жүҫеҲ°йӘҢиҜҒжҲҗеҠҹеҗҺеҸ‘йҖҒзҡ„иҜ·жұӮпјҲдҫӢеҰӮпјҡverifyгҖҒcheckгҖҒvalidateпјүгҖӮ
4. еңЁ **Payload** жҲ– **Response** дёӯжүҫеҲ°еҸӮж•° `sceneId`пјҲ`CaptchaSceneId` жҲ– `sId`пјүгҖӮ

![](./images/alibaba-task/sceneId.png)

иҜҘеҸӮж•°д№ҹеҸҜд»ҘйҖҡиҝҮеңЁзҪ‘з»ңиҜ·жұӮдёӯжҗңзҙўиҺ·еҸ–пјҡ

1. жү“ејҖеҢ…еҗ« CAPTCHA зҡ„йЎөйқўпјҢз„¶еҗҺиҝӣе…Ҙ **DevTools** вҶ’ **Network** ж ҮзӯҫгҖӮ
3. дҪҝз”ЁжҗңзҙўпјҲCtrl + FпјүжҹҘжүҫе…ій”®еӯ— `sceneId` жҲ– `CaptchaSceneId`гҖӮ

![](./images/alibaba-task/sceneId1.png)


### `prefix`

`prefix` еҸҜд»Ҙд»Һз”ЁдәҺеҠ иҪҪ CAPTCHA д»»еҠЎж–Үжң¬зҡ„иҜ·жұӮ URL дёӯиҺ·еҸ–пјҡ

1. жү“ејҖеҢ…еҗ« CAPTCHA зҡ„йЎөйқўгҖӮ
2. жүҫеҲ°дёҺеҠ иҪҪд»»еҠЎзӣёе…ізҡ„иҜ·жұӮпјҲйҖҡеёёйҖҡиҝҮ **DevTools** вҶ’ **Network**пјүгҖӮ

### йӘҢиҜҒз ҒеҸӮж•°зҡ„жҸҗеҸ–дёҺеҮҶеӨҮ

жң¬з« иҠӮжҸҸиҝ°дәҶеңЁзӣ®ж ҮзҪ‘з«ҷдёҠжҸҗеҸ–еҝ…иҰҒеҸӮж•°гҖҒи§ЈеҶійӘҢиҜҒз Ғд»ҘеҸҠйҮҚж–°жү§иЎҢзҷ»еҪ•жҺҲжқғзҡ„е®Ңж•ҙжөҒзЁӢгҖӮ

1. еҲқе§Ӣзҷ»еҪ•иҜ·жұӮпјҡ

```http

````

```json

```

*иҜ·жұӮеӨҙзӨәдҫӢиҜ·еҸӮи§Ғз« иҠӮ [иҮӘеҠЁйӘҢиҜҒз Ғи§ЈеҶізӨәдҫӢ](#РҝСҖРёРјРөСҖСӢ-Р°РІСӮРҫРјР°СӮРёСҮРөСҒРәРҫРіРҫ-СҖРөСҲРөРҪРёСҸ-РәР°РҝСҮРё)*гҖӮ

2. жңҚеҠЎеҷЁе“Қеә”зұ»еһӢгҖӮжңҚеҠЎеҷЁеҸҜиғҪиҝ”еӣһдёӨз§Қз»“жһңпјҡ

* 2.2 еёҰйӘҢиҜҒз Ғзҡ„е“Қеә”пјҲжңҚеҠЎеҷЁиҝ”еӣһ HTML йЎөйқўиҖҢйқһ JSONпјүпјҡ
